About Code Diff Checker

A lightweight browser extension and standalone web tool for comparing two blocks of text or code, inspired by diffchecker.com.

⧉ Split view — Side-by-side with independent line numbers
≡ Unified view — Single column with +/− prefixes
⎘ Copy — Copy original text to clipboard
↑ Export — Download as standard .diff file
🔲 Ignore Whitespace — Skip indentation differences
🌙 Dark / Light mode
Version 1.0 · Pure HTML / CSS / JS — no build step
